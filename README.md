# Lifebell pm- Skills

서일이앤씨 AX 교육용 Claude Code 커스텀 스킬 모음입니다.

## 🚀 설치 방법 (수업 시간)

Claude Code에 아래 문장을 입력하면 모든 pm- 스킬이 자동으로 설치됩니다:

```
이 링크에서 pm-스킬 전부 설치해줘: https://github.com/mshsese/pm-skills
```

## 📋 스킬 목록

| 스킬 | 설명 |
|------|------|
| `/pm-caveman` | 토큰 낭비 없이 AI와 효율적으로 일하는 방법 |
| `/pm-deep-interview` | AI가 기획을 깊이 있게 인터뷰하고 요구사항을 정리 |
| `/pm-grill-me` | 기획을 집요하게 검증하고 문제점 도출 |
| `/pm-grill-with-docs` | 도메인 검증 + terms.md 용어 사전 자동 생성 |
| `/pm-write-plan` | 모호한 기획을 실행 가능한 프롬프트로 변환 |
| `/pm-requesting-code-review` | 개발 완료 후 코드 리뷰 요청 |
| `/pm-ctx-update` | 오늘 작업 내용을 ctx.md에 자동 정리 |
| `/pm-auto` | Claude Code 자율 실행 모드 온/오프 |
| `/pm-office-hours` | 업무 시간 회의 문서 작성 지원 |
| `/pm-shadcn` | shadcn/ui 컴포넌트 기반 개발 |
| `/pm-to-issues` | 기획을 GitHub Issue로 자동 변환 |

## 📖 사용 방법

설치 후 Claude Code에서 다음과 같이 스킬을 호출합니다:

```bash
/pm-deep-interview
# → AI가 당신의 기획을 깊이 있게 인터뷰

/pm-write-plan
# → 기획을 실행 가능한 단계로 변환

/pm-ctx-update
# → 오늘 작업을 ctx.md에 정리
```

## ℹ️ 참고사항

- 모든 스킬은 한국어를 기본으로 지원합니다.
- 스킬 업데이트는 GitHub repo를 최신으로 유지하면 자동 반영됩니다.
- 문제가 발생하면 Claude Code 설정에서 스킬 폴더를 확인하세요: `~/.claude/skills/`
